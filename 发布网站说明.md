# 发布网站说明

当前网站可以作为静态网站发布，公开访问后用户可直接在浏览器里使用：

- 上传单张纯色背景 UI 图快速拆分。
- 上传第 2 张不同底色图做双底色精算。
- 上传左右双底色图自动裁分拆解。
- 上传上下双底色图自动裁分拆解。
- 单个 PNG 下载。
- 全部 ZIP 下载。

## 当前公开版不包含

“AI 自动生成第 2 底色图”暂未接入公开服务。

原因：

- 这个能力需要 OpenAI API Key。
- API Key 不能放在前端网页里。
- 如果给所有用户开放，需要后端、账号计费、额度控制和防滥用。

当前按钮会提示功能暂未开放，后续可以再接后端。

## 最简单发布方式：Vercel

1. 打开 Vercel。
2. 新建项目。
3. 导入这个文件夹所在仓库，或上传项目文件。
4. Framework 选择 `Other` 或保持默认。
5. Build Command 留空。
6. Output Directory 留空或填 `.`。
7. 发布后会得到一个公网网址。

项目里已经包含：

```text
vercel.json
```

## 也可以用 Netlify

1. 打开 Netlify。
2. 选择 Add new site。
3. 上传项目文件夹或导入仓库。
4. Publish directory 填：

```text
.
```

项目里已经包含：

```text
netlify.toml
```

## 也可以用 GitHub Pages

1. 把当前项目上传到 GitHub 仓库。
2. 进入仓库 Settings。
3. 打开 Pages。
4. Source 选择当前分支。
5. Folder 选择 `/root`。
6. 保存后等待生成网址。

项目里已经包含：

```text
.nojekyll
```

## 需要发布的核心文件

至少需要：

```text
index.html
styles.css
app.js
vercel.json
netlify.toml
.nojekyll
```

其他 `.md` 说明文件可保留，也可发布前删除。
